# 🚀 iBoss - AI-Powered Project Management System  

**iBoss** is an AI-driven project management tool that helps developers efficiently manage their projects by assigning tasks, tracking progress, and providing AI-powered feedback on GitHub submissions.  

## 🌟 Features  
✅ **User Authentication** (Register/Login)  
✅ **Create & Manage Projects**  
✅ **Assign Tasks with Deadlines**  
✅ **Submit GitHub Repositories for Review**  
✅ **AI-Powered Code Feedback**  
✅ **Task Scoring System (Win & Lose Points)**  
✅ **ChatGPT-Like AI Sidebar for Developer Assistance**  
✅ **Dark Mode & Theme Toggle**  

## 🛠️ Tech Stack  
- **Frontend:** HTML, Tailwind CSS, JavaScript  
- **Backend:** Node.js (Express.js) / ASP.NET  
- **Database:** MongoDB / MySQL  
- **AI Integration:** OpenAI API for AI Assistance  

## 🎯 Getting Started  

### 1️⃣ **Clone the Repository**  
```bash
git clone https://github.com/your-username/iBoss.git
cd iBoss
```

### 2️⃣ **Install Dependencies**  
```bash
npm install  # For Node.js (Express.js backend)
```

### 3️⃣ **Run the Application**  
```bash
npm start   # Starts the backend server
```

### 4️⃣ **Open in Browser**  
Visit: `http://localhost:3000`  

## 📌 Usage  

### **📝 Creating a Project**  
1. Click the **"Create Project"** button.  
2. Fill in the project details (name, type, description, GitHub repo link, etc.).  
3. Submit the form to save the project.  

### **📌 Assigning & Completing Tasks**  
1. Click **"Assign Task"** to generate a detailed task with a deadline.  
2. Complete the task and submit your **GitHub repository link**.  
3. AI reviews your code and gives feedback (approval or suggested improvements).  
4. Earn or lose points based on code quality.  

### **🤖 AI Chat Assistant**  
- Click the 🤖 AI icon to open the **ChatGPT-like AI assistant**.  
- Ask coding questions or get guidance on tasks.
